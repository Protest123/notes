# Краткий итог по командам в цепочке
## `find ~ -name "*file*" -exec ls -lh {} \; | sort -k5 -hr | head -3`

- `find ~ -name "*file*"`  - Поиск файла - находит все файлы с `file` в имени в домашнее директории.
- `-exec ls -lh {} \;`  - Детализация - для каждого найденного файла выводит подробную информацию (`ls -lh`) 
- `sort -k5 -hr`  - Сортировка - сортирует строки по 5-му стлобцу (размер) от большего к меньшему (`-r`)с учётом единиц (`-h`)
- `head -3`  - Ограничение вывода - выводит только первые 3 строки результата

# Дополнительные полезные команды для работы с пайпами

- `tail -n` - выводит последние `n` строк (например, `tail -5` - последние 5 строк)
- `uniq` - удаляет дублирующиеся строки (часто используются после `sort`)
- `awk '{print $5}'` - извлекает определённый столбец (здесь - 5-й, размер файла)
- `cut -d' ' -f5` - аналогично `awk`, вырезает столбец по разделителю (здесь-пробел)

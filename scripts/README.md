# Шпаргалка по скриптам

## backup.sh
Создаёт архив папки ~/notes в ~/backups с датой в имени.
./backup.sh

## cleanup.sh
Удаляет временные файлы (.tmp, .save, .swp) в указанной папке.
./cleanup.sh                # в текущей папке
./cleanup.sh /путь/к/папке  # в другой папке

## greet.sh
Приветствует пользователя в зависимости от времени суток.
./greet.sh        # Гость
./greet.sh Миша   # по имени

## watchdog.sh
Проверяет, запущен ли процесс, и если нет — запускает.
./watchdog.sh sleep

## check.sh
Демонстрация условий if/elif/else и проверки файлов.
./check.sh hello

## loop.sh
Демонстрация циклов for (по словам, файлам, диапазону).
./loop.sh

## while.sh
Демонстрация цикла while со счётчиком.
./while.sh

## func.sh
Демонстрация функций в bash.
./func.sh

## hello.sh
Простой скрипт: приветствие, дата, текущая папка.
./hello.sh

## vars.sh
Демонстрация переменных и аргументов скрипта.
./vars.sh аргумент1 аргумент2

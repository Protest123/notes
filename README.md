## Git: основные команды (шпаргалка)

| Команда | Что делает |
|---------|-------------|
| `git status` | проверить состояние |
| `git add <файл>` | добавить в staging |
| `git commit -m "..."` | сохранить коммит |
| `git log --oneline` | история коммитов |
